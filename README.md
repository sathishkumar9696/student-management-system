# Student Management System

## 📌 Description
A Java console-based application to manage student records using CRUD operations.

## 🚀 Features
- Add Student
- View Students
- Delete Student
- Search Student

## 🛠️ Technologies Used
- Java
- ArrayList
- OOP Concepts

## ▶️ How to Run
1. Compile Main.java
2. Run the program
3. Use menu options

## 📂 Project Structure
- Main.java
- Student.java
